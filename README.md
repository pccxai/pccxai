# PCCX AI

**Independent Hardware/Software Research Project** by Hyunwoo Kim [](https://github.com/hwkim-dev)

From NPU core architecture to a full AI-powered development toolchain, FPGA porting, and LLM integration — building a complete end-to-end HW/SW ecosystem.

### 📍 Repositories

- **[pccx](https://github.com/pccxai/pccx)**  
  SystemVerilog-based NPU Core Architecture (RTL)

- **[pccx-lab](https://github.com/pccxai/pccx-lab)**  
  **Rust + Tauri All-in-One Development & Verification Toolchain**  
  An AI-powered comprehensive platform that handles **design, development, verification, performance/load analysis, bottleneck detection, visualization, automated reports, documentation, and full pipeline orchestration**.  
  Future plans include LLM integration for automatic optimal ISA combination generation and even end-to-end SystemVerilog design assistance.

- **[pccx-FPGA-NPU-LLM-kv260](https://github.com/pccxai/pccx-FPGA-NPU-LLM-kv260)**  
  Xilinx KV260 board integration with NPU + LLM (KV-cache) implementation and documentation

### 🚀 Current Status
- NPU Core: RTL design completed
- pccx-lab: AI Agent / Chat / Custom AI + Visualization + Report Pipeline in active development
- FPGA: KV260 porting and LLM integration in progress

**Solo research project** — open to contributions and collaboration.
